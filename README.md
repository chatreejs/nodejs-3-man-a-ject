## Requirement

- node.js

## Usage

Clone it!

```
$ git clone https://gitlab.com/n-man-a-ject/3-man-a-ject
```

Go into project directory and run the command

```
$ npm i
$ node server.js
```

Open `http://localhost:3000` and enjoy!
